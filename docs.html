Propósito

🎯 Objetivo do Projeto

O Manager Project N8N foi criado para centralizar a gestão de infraestrutura de automação. Ele permite que a equipe visualize em tempo real o estado de saúde, stack tecnológica e acessos de todos os nossos projetos ativos.

Monitoramento

Estados Online, Construção ou Manutenção atualizados via webhook.

Transparência

Descrições técnicas e links diretos para ambientes de produção e staging.

Infraestrutura

🏗️ Arquitetura Técnica

Nossa solução utiliza um stack moderno, sem cabeçalhos pesados, focada em estabilidade e agilidade.

Motor n8n

Processa lógica de negócio (`prod-api-manager-project`), valida dados e expõe os endpoints REST.

Banco Supabase

Armazenamento resiliente utilizando PostgreSQL para persistência dos metadados dos projetos.

Frontend Glass

Interface SPA (Single Page Application) em JS puro, consumindo Webhooks em tempo real.

Desenvolvimento

🚀 APIs e Integrações

Utilize os endpoints abaixo para gerenciar os dados do portfólio.

Consulta de Projetos (GET)

Retorna a lista completa de projetos em formato JSON.

GET https://agentes-n8n.cb16s5.easypanel.host/webhook/prod-api-manager-project

Cadastro de Novo Projeto (POST/Form)

Envie dados via formulário estruturado do n8n (Título, Status, URL, Descrição, Tags, **Registrar API**, **Descrição da API**).

POST https://agentes-n8n.cb16s5.easypanel.host/form/ce79d13d-120a-4ac5-bf96-ae94401c6119
Operação

🛠️ Manutenção

Para adicionar novos projetos, acesse o formulário de gestão interno. Todas as alterações refletem instantaneamente no dashboard principal através do sistema de cache resiliente da Easypanel.
